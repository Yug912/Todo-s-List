# ✅ TaskFlow – Todo List App

A modern **Todo List Application** built using **React.js, Bootstrap, and Local Storage** that helps users manage daily tasks efficiently.

Add, organize, and track your tasks with a clean interface while ensuring your data remains saved even after refreshing or reopening the browser.

## 🎯 How It Works

1. Enter a Todo Title.
2. Add a Todo Description.
3. Click the **Submit** button.
4. The task gets added to the Todo List.
5. All todos are automatically saved in Local Storage.
6. Refreshing or reopening the browser preserves your tasks.
7. Delete tasks whenever they are completed.

---

## 💾 Local Storage Functionality

This application uses the browser's **Local Storage API** to save user data.

### Benefits

- Data remains available after page refresh.
- No backend or database required.
- Fast and lightweight storage solution.
- Provides a better user experience by preventing data loss.

### Workflow

```text
User Adds Todo
       ↓
React State Updates
       ↓
Local Storage Updates
       ↓
Page Refresh
       ↓
Todos Loaded From Local Storage
```
